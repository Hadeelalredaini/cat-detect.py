# 🐾 Cat Detection Project

Welcome to my **Cat Detection** project! This is a fun computer vision project designed to identify cats in images using the power of Python and OpenCV. 🐈✨

## 💡 Why this project?
I chose to implement this project to explore the basics of **Object Detection** within the AI field. 

**Note on detection:**
In this implementation, the model is specifically trained to identify the **nose and mouth area** of the cat. It is a great starting point for understanding how Haar Cascades work and how models are trained to recognize specific features.
